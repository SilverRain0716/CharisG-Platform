## 3. 화면별 상세

### 3.1 대시보드 (홈)

| 위젯 | 구성 요소 |
|------|----------|
| 파이프라인 퍼널 | CJ Total(38K)→Collected(~6.2K)→Hard Filter(335)→GO(107)→Listed(42)→Active. 전환율 |
| 계정 건강도 게이지 | ODR(<0.5%), LSR(<2%), Cancel(<1%), VTR(>99%). 아마존 기준 대비 게이지. Phase 0: 수동 입력 뱃지 |
| 핵심 KPI | 활성 상품, 총 매출, 평균 마진, 주문 처리율 |
| 알림 피드 | CJ 품절, 가격 변동, 계정 건강도 경고 |

### 3.2 스코어링

**3×3 매트릭스 히트맵**: Demand(Y) × Margin(X). 셀 내 상품 수 + GO 비율. AA/AB=녹, BA/BB=파, BC/CA=주, CB/CC=빨. 셀 클릭 → 필터링.

**스코어 분포**: Demand/Gap/Margin 각각 히스토그램.

**GO/SKIP 테이블**: DataTable. 상품명, 카테고리, CJ가격, Amazon p75, 실질마진, D/G/M, 매트릭스, sort_score(기본 내림차순), GO 판정, 상태. 일괄 상태 변경 + CSV.

**스코어링 실행**: 수동 트리거 + 진행률 + 이력.

**Hard Filter 탈락 분포**: 사유별(8개) 바 차트.

### 3.3 상품 관리

**칸반 트래커**: Candidate→Listed→Active→Paused. 드래그앤드롭. 카드: 이미지, 상품명, 마진율, [매트릭스] 뱃지, Tier.

**가격 경쟁력 산점도**: X=Amazon p75, Y=CJ 가격. 대각선=손익분기. 아래=경쟁력(녹), 위=고가(빨). 호버 팝업.

**상품 상세**: 타임라인(CJ수집→Filter→스코어링→GO→Listed→Active). 기본정보, 스코어 상세, 가격 이력, 판매 성과(Phase 1+).

**리스팅 콘텐츠 편집**: Tier 1(풀 최적화) vs Tier 2(기본). 타이틀, Bullets, 설명, 키워드, 이미지.

### 3.4 매출·성과 (Phase 1+ placeholder)

Phase 0: "Phase 1 도달 시 활성화" 메시지 + 목표 $500 프로그레스. 빈 차트 프레임.
Phase 1+: 일/주/월 매출, 상품별 P&L(Revenue-COGS-Shipping-Fee-Ad=Net), Top/Bottom 랭킹.

### 3.5 모니터링

**CJ 재고·품절**: GO 상품 재고량. <10 경고, 품절 시 Paused 제안.

**가격 변동**: Listed/Active Amazon p75 추적. 임계치 초과 → 재가격 제안 → 승인 자동 수정.

**계정 건강도 상세**: 4개 지표 시계열. 아마존 기준 + 내부 목표 라인.

### 3.6 설정

스코어링 파라미터(Hard Filter 8개 임계값, 가중치), 크롤러 설정(프록시, 딜레이), API 연동(CJ, SP-API Phase 1+, pytrends), 알림(Discord 웹훅 + on/off), 계정 건강도 수동 입력 폼.
