## 3. 화면별 상세

### 3.1 대시보드 (홈)

| 위젯 | 구성 요소 |
|------|----------|
| 파이프라인 퍼널 | 키워드→소싱→마진→통관→GO→등록→업로드→활성 단계별 건수 + 전환율. 클릭 → 해당 화면 이동 |
| 오늘의 할 일 | GO/NO-GO 대기, 업로드 대기, 미처리 CS 건수 |
| 핵심 KPI | 활성 상품 수, 월 매출, 평균 마진율, 주문 처리율 |
| 알림 피드 | 가격 변동, 품절, CS 접수 시간순 |

### 3.2 디스커버리

**키워드 현황**: 네이버 데이터랩 + 검색광고 API 기반. 대표 키워드, 클러스터 그룹, 검색량, 트렌드, 상태. 주 1회 자동 + 수동 버튼. **풀 파이프라인 실행 버튼** (데이터랩→검색량→클러스터링→상위50 일괄 + 단계별 진행률).

**트렌드 분석**: Google Trends + 네이버 검색량 병행 차트. 카테고리별 demand map.

**소싱 전환 현황** (Phase 2+): 전환율 추적 백엔드 구현 후 활성화.

### 3.3 소싱

**소싱 후보 리스트**: ASIN, 상품명, 아마존 가격, 평점, 리뷰수, 재고상태, CJ 필터, **배송제한(PASS/WARN/REJECT)**. 필터/정렬 + 일괄 선택 → 마진 분석 전송.

**마진 분석**: 셀러 마진(아마존 가격 + 배대지 수수료 + 반품 충당금 + CS비 → 순익), 고객 총 비용(판매가 + 배송비 + 예상 관부가세). **경쟁 가격 비교 위젯**: 고객 총 비용 vs 쿠팡/네이버 경쟁사 + 경쟁력(HIGH/MED/LOW).

**통관 체크**: 목록통관 vs 일반통관 자동 태그. 리스크 플래그. REJECT → GO/NO-GO 이전 자동 제외.

**GO/NO-GO 판단 테이블**: 30~100건 배치. 스코어링 = **Demand × Margin 종합 점수** (통관리스크는 별도 필터, 곱셈 아님). 일괄 GO/NO-GO + 사유 입력. 필터 프리셋: 고마진 우선, 통관 안전, 미판단 전체.

### 3.4 상품 관리

**상세페이지 제작**: AI 생성(Gemini 번역 + SEO 리라이팅 → 13섹션) → 편집 → 미리보기 → 확정. 템플릿 관리.

**등록 준비**: 체크리스트(상세페이지, 이미지, 카테고리, 필수필드). 검수 완료 → 업로드 큐.

**업로드 큐**: 대기 상품 + 대상 마켓 선택. 상태(대기/업로드중/성공/실패). 수동 트리거.

**활성 상품 목록**: 상품명, 판매채널, 판매가, 마진율, BSR, 주문수, 상태.

**상품 상세**: 진행 타임라인(키워드→소싱→GO→등록→판매중). 기본/수익/운영 이력.

### 3.5 주문·CS

**주문 현황**: 6단계 칸반(접수→아마존구매→배대지→국제배송→국내배송→완료). 지연 건 하이라이트.

**배송 추적**: 구현 완료 (CJ 자동 + 수동 입력). 향후 UNIPASS/택배사 API 확장.

**CS 티켓**: 상태/유형/우선순위. AI 초안 자동 생성 → 수동 확인 발송.

**반품·환불**: 접수→사유확인→환불 워크플로우. 반품 정책 자동 적용.

### 3.6 모니터링

**가격 변동**: 아마존 가격 추이 + 임계치 초과 시 재가격 제안 → 승인 → 자동 수정.

**재고·품절**: 정기 크롤링 → 품절 시 알림 + 리스팅 비활성화 제안.

**경쟁 가격 비교**: 내 판매가 vs 쿠팡/네이버 경쟁사 + 경쟁력 점수 추이.

### 3.7 설정

**마진 파라미터**: 배대지 수수료, 반품 충당금, CS비, 환율 소스.

**크롤링 스케줄**: 네이버 데이터랩, 아마존, 경쟁사 주기 + 이력.

**알림 채널**: Discord 웹훅 (구현 완료) + 추후 Slack/이메일 확장.

**API 연동**: 네이버 커머스(완성), 쿠팡(완성), 검색광고, **관세 자동화 (AI HS코드 + 관세청 DB 12,469건 완료, 간이세율 폴백)**.
